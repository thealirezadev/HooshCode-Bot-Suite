# فونت فارسی برای نمودار هفتگی بازار

برای اینکه متن‌های فارسی در نمودار هفتگی (`utils/chart.py`) به‌درستی
(بدون باکس خالی یا حروف جداافتاده) نمایش داده شوند، فونت رایگان و
متن‌باز **Vazirmatn** را دانلود و در همین پوشه قرار دهید:

1. به آدرس زیر بروید:
   https://github.com/rastikerdar/vazirmatn/releases

2. فایل `Vazirmatn-Regular.ttf` را دانلود کنید.

3. آن را دقیقاً با همین نام در این مسیر قرار دهید:
   `assets/fonts/Vazirmatn-Regular.ttf`

اگر این فایل وجود نداشته باشد، کد به‌صورت خودکار به دنبال فونت‌های
فارسی‌خوان نصب‌شده روی سیستم (مثل Noto Naskh Arabic یا Tahoma) می‌گردد.
در صورت نبود هیچ‌کدام، یک هشدار در کنسول چاپ می‌شود و ممکن است حروف
فارسی به‌درستی نمایش داده نشوند.
